## What I work on

**Data Domains**
- Retail & e-commerce analytics, demand forecasting, competitor & SKU performance
- Customer experience: survey pipelines (Medallia, Mopinion), NPS & satisfaction analytics
- Marketing intelligence: DV360, Ahrefs, SimilarWeb ingestion & reporting
- HR & workforce planning data, logistics & returns (PostNL), weather-demand modelling (KNMI)

**Cost & Performance**
- Migrated full-refresh dbt models to incremental — significant reduction in Databricks DBU and S3 costs
- Spark tuning: partition pruning, broadcast joins, AQE to eliminate data skew, small file compaction
- Automated Delta table OPTIMIZE + VACUUM with Z-ordering on high-cardinality columns

**Data Quality**
- pytest suites for all Python pipeline modules — runs on every PR in CI
- dbt schema tests, custom generic tests, and source freshness checks
- Automated alerting on pipeline failures before issues reach stakeholders

**Team & Platform**
- Onboarded data analysts to the dbt repo — Git workflows, model conventions, writing tests independently
- Built GitHub Actions CI/CD: Ruff linting, pytest, dbt compile & test — PRs blocked on failure
- Built AI agents on Vertex AI ADK + MCP for HR, procurement, returns, and Jira use cases
